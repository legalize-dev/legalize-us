# § 60902. Application

##### § 60902. Application

Each agency shall develop a written application to be used for designated persons to request a case file review under [section 60901 of this title](/us/usc/t34/s60901).

*([Pub. L. 117–164, § 3](/us/pl/117/164/s3), Aug. 3, 2022, [136 Stat. 1359](/us/stat/136/1359).)*
