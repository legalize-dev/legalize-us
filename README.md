# legalize-us

> **Work in progress.** This repository is under active development. Structure, filenames, and content may change without notice.

The complete [United States Code](https://uscode.house.gov/) as Markdown files — every section of federal law in a git repository.

## What's here

- **60,000+ sections** of the US Code across 53 titles
- Each section is one `.md` file with YAML frontmatter (title, identifier, status, source URL, source credit)
- Text preserves the official structure: subsections, paragraphs, clauses, cross-references, editorial notes, and statutory notes
- Source: [Office of the Law Revision Counsel (OLRC)](https://uscode.house.gov/) — the official publisher of the US Code

## File naming

Each file follows the pattern `us/USC-T{title}-S{section}.md`:

```
us/USC-T18-S1341.md    → 18 U.S.C. § 1341 (Mail fraud)
us/USC-T47-S230.md     → 47 U.S.C. § 230 (Section 230)
us/USC-T26-S501.md     → 26 U.S.C. § 501 (Tax-exempt organizations)
```
